Features

  • Allows the storage of argument information
  • Allows default values for optional arguments if the arguments are not provided
  • Allows optional arguments to be mixed with positional arguments in any order
  • Allows optional arguments to serve as flags
  • Allows short name options for optional arguments, in addition to longer names
  • Throws exception if specified argument does not exist
  • Allows names optional arguments with single values, which may have help and datatype information
  • Allows the inclusion of additional descriptive information on the program and each argument and provide named "-h" argument that shows usage and help information
  • Allows datatype information to be added to arguments so that non-string arguments can be used
  • Allows optional arguments to be specified as required arguments
  • Allows arguments to have a restricted set of possible choices for its values
  • Allows arguments information to be loaded from an XML file
  • Allows only string-valued positional arguments and retrieve them from the command-line
  • Allows argument information to be saved to an XML file
